## 💡 어떤 문제를 풀었나요?

15번 요세푸스 문제

## 🔑 풀이 접근 방식

1. n을 기준으로 queue 배열을 만든다. (1~n)
2. queue에 1명 초과로 남아있는 동안 반복
3. k-1번 동안 맨 앞 사람을 꺼내서 맨 뒤로 보낸다.
4. 맨 앞 사람을 제거한다. (k번째 사람 탈락)
5. queue에 마지막으로 남은 사람의 번호를 반환한다.
